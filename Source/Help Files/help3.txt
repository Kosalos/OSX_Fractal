3 Lights

Light#:  select which light to alter (three are available)

Bright: set light brightness
Spread: set how spread out the light effect is
X,Y,Z: set position of light
R,G,B: set light color

Fog Amount: set stength of Fog effect
R,G,B: set fog color

O Strength: set strength of Orbit trap effect
Cycles: set #cycles of the obit trap algorithm
X,Y,Z,R Weight: set strength of the 4 orbit trap parameters
X,Y,Z,R Color: select palette color for respctive orbit paramter
Fixed Trap (0,1,2):  set style of orbit trap seed point (0 = legacy style)
X,Y,Z: set position of orbit trap for styles 1 and 2
